The game management system allows administrators to maintain the board game catalog, including browsing by category and adding new games.

## Accessing Game Management

<Steps>
  <Step title="Open Admin Dashboard">
    Navigate to the admin menu at `/app/MenuAdmin.html`
  </Step>
  <Step title="Select Juegos Section">
    Click the **Juegos** button to reveal the game catalog interface
  </Step>
</Steps>

## Game Catalog Structure

The game catalog is organized with a two-column layout:

```html
<div id="Div1" class="SuperPuesto">
    <div class="col-3">Categoria</div>
    <div class="col-9">Lista de Juegos</div>
</div>
```

- **Left Column (25%)**: Category filter navigation
- **Right Column (75%)**: Game display grid

## Category Filtering

The user interface implements dynamic category filtering using jQuery:

```javascript
// Filter products by category
$('.category_item').click(function(){
    var catProduct = $(this).attr('category');
    
    // Hide all products with animation
    $('.product-item').css('transform', 'scale(0)');
    setTimeout(function(){
        $('.product-item').hide();
    }, 400);
    
    // Show filtered products
    setTimeout(function(){
        $('.product-item[category="'+catProduct+'"]').show();
        $('.product-item[category="'+catProduct+'"]').css('transform', 'scale(1)');
    }, 400);
});
```

This creates a smooth animation effect when switching between categories.

## Viewing All Games

Click the **"Todo"** category to display all games:

```javascript
$('.category_item[category="all"]').click(function(){
    setTimeout(function(){
        $('.product-item').show();
        $('.product-item').css('transform', 'scale(1)');
    }, 400);
});
```

## Game Display Format

Each game in the catalog is displayed as a product item with:

- Game image thumbnail
- Game title as clickable link
- Modal dialog for detailed information

## Adding New Games

<Steps>
  <Step title="Click Añadir Juegos">
    Select the **Añadir Juegos** button in the admin dashboard
  </Step>
  <Step title="Access Add Game Form">
    The interface reveals the game addition section:
    ```html
    <div id="Div4" class="Superpuesto Oculto">
        Opciones para añadir juegos por categoria y tales :P
    </div>
    ```
  </Step>
  <Step title="Enter Game Details">
    Fill in the required information including:
    - Game title
    - Category selection
    - Game image
    - Description
  </Step>
</Steps>

<Note>
The add game functionality is managed through the `RevelarDiv4()` function which toggles the visibility of the game addition form.
</Note>

## Game Modal Dialogs

Clicking on any game opens a modal with detailed information:

```html
<div id="modal1" class="modalmask">
    <div class="modalbox movedown">
        <a href="#close" title="Close" class="close">X</a>
        <h2>Game Title</h2>
        <p>Detailed game description</p>
        <p>Additional game information</p>
    </div>
</div>
```

Different modal animation effects are used:
- `movedown`: Slides down from top
- `rotate`: Rotates into view
- `resize`: Scales up from center

<Warning>
Ensure all game images are properly stored in the `/img/Juegos/` directory structure organized by category before adding them to the catalog.
</Warning>
