The event management system allows administrators to create and manage gaming events, while providing users with visibility into upcoming activities.

## Accessing Event Management

<Steps>
  <Step title="Open Admin Dashboard">
    Navigate to `/app/MenuAdmin.html`
  </Step>
  <Step title="Select Eventos Section">
    Click the **Eventos** button to access event management
  </Step>
</Steps>

## Event Interface

The events section is controlled by the `RevelarDiv2()` function:

```javascript
function RevelarDiv2(){
    Juegos.className=Juegos.className+" Oculto";
    Eventos.className="Superpuesto";
    Usuarios.className=Usuarios.className+" Oculto";
    InsercionDeJuegos.className=InsercionDeJuegos+" Oculto";
}
```

### Admin Event View

In the admin dashboard:

```html
<div id="Div2" class="Superpuesto Oculto">
    Eventos proximos
</div>
```

### User Event View

Regular users see events in their dashboard:

```html
<div id="Div2" class="Oculto">
    <h6>Eventos proximos</h6>
</div>
```

## Event Visibility Toggle

The interface uses CSS class manipulation to show/hide event sections:

### Admin View Toggle

```javascript
var Eventos=document.getElementById('Div2');

function RevelarDiv2(){
    Eventos.className="Superpuesto"; // Make visible
    // Hide other sections
}
```

### User View Toggle

Similar functionality for regular users viewing events:

```javascript
function RevelarDiv2(){
    // Show events, hide games
}
```

<Warning>
When switching between sections, ensure you've saved any changes as the visibility toggle doesn't preserve unsaved form data.
</Warning>

## User Event Participation

Users can interact with events through:

### Event Discovery

1. **Browse Events**: View upcoming events in the Eventos section
2. **Check Calendar**: Reference the monthly calendar for dates
3. **Event Details**: Click on events for more information

### Event Types

Different event types serve different purposes:

| Event Type | Description | Typical Duration |
|------------|-------------|------------------|
| Game Night | Regular gaming sessions | 3-4 hours |
| Tournament | Competitive events | Full day |
| Workshop | Learning new games | 2-3 hours |
| Campaign | Ongoing RPG sessions | Weekly/Bi-weekly |
| Convention | Large community gathering | Multiple days |

## Event Workflow

### For Administrators

1. Access admin dashboard
2. Click **Eventos** button
3. Create new event
4. Set event details
5. Publish event
6. Monitor registrations
7. Send reminders
8. Manage event day activities

### For Users

1. Navigate to user menu
2. Click **Eventos** button  
3. Browse upcoming events
4. View event details
5. Register for events
6. Receive notifications
7. Attend events

<Note>
The events interface uses the same JavaScript toggle system (`appMenu.js`) as other dashboard sections, ensuring consistent behavior across the platform.
</Note>
