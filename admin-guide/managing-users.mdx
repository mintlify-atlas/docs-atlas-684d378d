The user management system provides administrators with tools to view, monitor, and manage user accounts on the Pacto Lúdico platform.

## Accessing User Management

<Steps>
  <Step title="Navigate to Admin Dashboard">
    Access the admin menu at `/app/MenuAdmin.html`
  </Step>
  <Step title="Click Consultar Usuarios">
    Select the **Consultar Usuarios** button to reveal the user management interface
  </Step>
</Steps>

## User Management Interface

The user management section is toggled using the `RevelarDiv3()` function:

```javascript
function RevelarDiv3(){
    Juegos.className=Juegos.className+" Oculto";
    Eventos.className=Eventos.className+" Oculto";
    Usuarios.className="Superpuesto";
    InsercionDeJuegos.className=InsercionDeJuegos.className+" Oculto";
}
```

When activated, it displays:

```html
<div id="Div3" class="Superpuesto Oculto">
    Lista de Usuariooos
</div>
```

## Profile Tab Navigation

The profile interface uses the same toggle system as the admin dashboard:

```javascript
function RevelarDiv1(){ // Show Information
    // Information tab becomes visible
}

function RevelarDiv2(){ // Show Notifications
    // Notifications tab becomes visible
}
```

Profile buttons:
```html
<button onclick="RevelarDiv1()" class="boton">Información</button>
<button onclick="RevelarDiv2()" class="boton">Notificaciones</button>
```

## User Navigation Options

Both regular users and administrators can navigate between views:

```javascript
function CambiarPagina(){
    var text = select.options[select.selectedIndex].innerText;
    if(text=="Perfil"){
        window.location='../app/Perfil.html';
    }else if(text=="Menú"){
        window.location='../app/MenuUser.html';
    }
}
```

The dropdown selector provides quick access:
- **Perfil**: Navigate to user profile page
- **Menú**: Return to main menu (User or Admin depending on role)

<Note>
The `CambiarPagina()` function is triggered when users select an option and click the navigation button.
</Note>

## User vs Admin Menu

The system maintains separate interfaces for different user roles:

| Feature | Admin Menu | User Menu |
|---------|------------|------------|
| File | `/app/MenuAdmin.html` | `/app/MenuUser.html` |
| Label Display | "Administrador" | "Usuario" |
| Game Management | Full access | View only |
| Event Management | Create/Edit | View only |
| User Management | Full access | Own profile only |
| Add Games | Yes | No |
