The Events section of Pacto Lúdico helps users discover upcoming gaming sessions, tournaments, and community gatherings.

## Accessing Events

Events are accessible from the main user menu alongside the game catalog:

<Steps>
  <Step title="Navigate to Main Menu">
    After logging in, you'll see the main menu with two primary options: Juegos (Games) and Eventos (Events).
  </Step>
  
  <Step title="Click Events Button">
    Click the "Eventos" button to reveal the events section:
    
    ```html
    <button onclick="RevelarDiv2()" class="boton">Eventos</button>
    ```
  </Step>
  
  <Step title="View Event Listings">
    Browse upcoming gaming events and community activities.
  </Step>
</Steps>

## Event Interface

The events section is implemented as a toggleable view in the main menu:

```html
<div class="row">
    <button onclick="RevelarDiv1()" class="boton">Juegos</button>
    <button onclick="RevelarDiv2()" class="boton">Eventos</button>
</div>

<!-- Events section -->
<div id="Div2" class="Oculto">
    <h6>Eventos proximos</h6>
</div>
```

## Section Switching Logic

The platform uses JavaScript to toggle between the Games catalog and Events view:

```javascript
var Juegos = document.getElementById('Div1');
var Eventos = document.getElementById('Div2');

function RevelarDiv1(){
    Juegos.className = "Superpuesto";
    Eventos.className = Eventos.className + " Oculto";
}

function RevelarDiv2(){
    Juegos.className = Juegos.className + " Oculto";
    Eventos.className = "Superpuesto";
}
```

<Note>
  The events section becomes visible when clicked, while the games catalog is hidden, providing a clean, focused view.
</Note>

## Event Information

Each event listing typically includes:

<Accordion title="Event Details">
  **Essential Information**:
  - Event title and description
  - Date and time
  - Location or venue
  - Game(s) to be played
  - Number of players needed
  - Skill level requirements
  - Registration status
  
  **Additional Details**:
  - Event organizer
  - Duration estimate
  - Required materials
  - Special instructions
