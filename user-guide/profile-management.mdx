The Profile section allows users to manage their personal information and view notifications from the Pacto Lúdico platform.

## Accessing Your Profile

The profile page is accessible through the navigation menu after logging in:

<Steps>
  <Step title="Navigate to Profile">
    Access the profile through the dropdown menu in the navigation bar:
    
    ```html
    <ul class="nav">
        <li><a href="" onclick="CambiarPagina()">Perfil</a>
            <ul>
                <li><a href="">Configuración</a></li>
                <li><a href="">Datos personales</a></li>
            </ul>
        </li>
    </ul>
    ```
  </Step>
  
  <Step title="Select Option">
    Choose between viewing your profile information or switching back to the main menu using the dropdown selector.
  </Step>
</Steps>

## Section Switching Logic

The profile page uses JavaScript to toggle between Information and Notifications:

```javascript
var Juegos = document.getElementById('Div1');
var Eventos = document.getElementById('Div2');

function RevelarDiv1(){
    Juegos.className = "Superpuesto";
    Eventos.className = Eventos.className + " Oculto";
}

function RevelarDiv2(){
    Juegos.className = Juegos.className + " Oculto";
    Eventos.className = "Superpuesto";
}
```

<Note>
  Only one section is visible at a time, with the active section showing the "Superpuesto" class and the inactive section receiving the "Oculto" (hidden) class.
</Note>

### Page Switching Function

```javascript
function CambiarPagina(){
    var text = select.options[select.selectedIndex].innerText;
    if(text == "Perfil"){
        window.location = '../app/Perfil.html';
    } else if(text == "Menú"){
        window.location = '../app/MenuUser.html';
    }
}
```
