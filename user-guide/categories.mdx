## How Filtering Works

When you click a category, the system filters games using jQuery:

```javascript
$('.category_item').click(function(){
    var catProduct = $(this).attr('category');
    
    // Mark selected category as active
    $('.category_item').removeClass('ct_item-active');
    $(this).addClass('ct_item-active');
    
    // Hide all games
    $('.product-item').css('transform', 'scale(0)');
    $('.product-item').hide();
    
    // Show only games in selected category
    $('.product-item[category="'+catProduct+'"]').show();
    $('.product-item[category="'+catProduct+'"]').css('transform', 'scale(1)');
});
```

<Note>
  The filtering system uses CSS transforms for smooth animations, scaling games down when hiding and up when showing.
</Note>

## Using Categories

<Steps>
  <Step title="View All Categories">
    The category navigation appears above the game grid in the Juegos section.
  </Step>
  
  <Step title="Select a Category">
    Click any category name to filter the collection.
  </Step>
  
  <Step title="Visual Feedback">
    The selected category receives the `ct_item-active` class for visual indication.
  </Step>
  
  <Step title="View Results">
    Games animate smoothly, showing only those matching the selected category.
  </Step>
</Steps>

<Tip>
  Click "Todo" (All) at any time to return to viewing the complete collection.
</Tip>
