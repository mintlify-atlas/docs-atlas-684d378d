### Modal Windows

CSS-only modals using `:target` pseudo-class:

```css
.modalmask {
  position: fixed;
  color: white;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  background: rgba(0,0,0,0.8);
  z-index: 99999;
  opacity: 0;
  -webkit-transition: opacity 400ms ease-in;
  -moz-transition: opacity 400ms ease-in;
  transition: opacity 400ms ease-in;
  pointer-events: none;
}

.modalmask:target {
  opacity: 1;
  pointer-events: auto;
}
```

**How it works:**
1. Modals are invisible by default (`opacity: 0`, `pointer-events: none`)
2. Clicking a link with `href="#modal1"` triggers `:target`
3. Modal fades in with opacity transition
4. Close button links to `href="#close"` to untarget

## Common Patterns

### Show/Hide Content

```javascript
function RevelarDiv1() {
  Juegos.className = "Superpuesto";
  Eventos.className = Eventos.className + " Oculto";
}
```

Toggle between `.Superpuesto` (visible overlay) and `.Oculto` (hidden).

### Category Filtering

JavaScript dynamically shows/hides products based on category:

```javascript
// Implemented in script.js
$('.category_item').click(function() {
  var category = $(this).attr('category');
  
  $('.product-item').each(function() {
    if (category == 'all' || $(this).attr('category') == category) {
      $(this).show();
    } else {
      $(this).hide();
    }
  });
});
```

## Customization Guide

<Steps>
  <Step title="Change Color Scheme">
    Update primary colors in CSS files:
    
    ```css
    /* Find and replace */
    #2D3E50 → Your primary color
    #E84C3D → Your accent color
    ```
  </Step>

  <Step title="Modify Layout">
    Adjust grid percentages:
    
    ```css
    .category_list { width: 20%; }  /* Sidebar */
    .products-list { width: 80%; }  /* Main area */
    ```
  </Step>

  <Step title="Update Typography">
    Change font family globally:
    
    ```css
    * {
      font-family: 'Your Font', Arial, sans-serif;
    }
    ```
  </Step>

  <Step title="Add Custom Components">
    Follow the naming convention:
    
    ```css
    .component-name { }
    .component_variant { }
    ```
  </Step>
</Steps>
