## Overview

Pacto Lúdico follows a simple, flat directory structure optimized for Firebase Hosting. The project is organized into logical folders for HTML pages, styles, scripts, and assets.

## Directory Layout

```
Pacto-Ludico/
├── app/                    # Application pages
│   ├── MenuAdmin.html     # Administrator menu
│   ├── MenuUser.html      # User menu with game catalog
│   └── Perfil.html        # User profile page
├── css/                    # Stylesheets
│   ├── styleIndex.css     # Login page styles
│   ├── styleMenu.css      # Menu and catalog styles
│   └── stylePerfil.css    # Profile page styles
├── img/                    # Images and assets
│   ├── Juegos/            # Game images by category
│   │   ├── Ameritrash/
│   │   ├── Cartas/
│   │   ├── Dados/
│   │   ├── Eurogames/
│   │   ├── Fichas/
│   │   ├── Filler/
│   │   ├── Guerra/
│   │   ├── Miniatura/
│   │   ├── Party Games/
│   │   ├── Rol/
│   │   └── Tablero tradicional/
│   ├── BOTON_INICIARSESION.png
│   ├── Calendario-Mensual-Septiembre-2019.jpg
│   ├── Usuario.png
│   └── fondito1.jpg       # Background image
├── js/                     # JavaScript files
│   ├── appIndex.js        # Login logic
│   ├── appMenu.js         # Menu and navigation logic
│   ├── jquery-3.2.1.js    # jQuery library
│   └── script.js          # Product filtering script
├── index.html              # Login page (entry point)
├── Registrar.html          # Registration page
├── 404.html                # Custom error page
├── firebase.json           # Firebase hosting config
└── .firebaserc             # Firebase project config
```

## CSS Organization

Each HTML page has its own stylesheet for separation of concerns:

### styleIndex.css
Login page styles including background, form elements, and error states.

**Key Classes:**
- `.Error` - Red border for invalid inputs
- `.ErrorLabel` - Brown text for error messages
- `.LabelUsado` - White text for active labels

### styleMenu.css
Comprehensive styles for the main application interface.

**Key Components:**
- Navigation menu (`.nav`)
- Category list (`.category_list`)
- Product grid (`.products-list`, `.product-item`)
- Modal windows (`.modalmask`)
- Responsive layout with Flexbox

### stylePerfil.css
Profile page specific styles.

## Image Assets

<Note>
  Game images are organized in subdirectories by category (Ameritrash, Cartas, Dados, etc.) under `img/Juegos/`. This structure mirrors the category filtering system in the UI.
</Note>

### Asset Categories

- **UI Elements**: Buttons, user icons, backgrounds
- **Game Images**: Organized by 12 game categories
- **Calendar**: Monthly event calendar image
