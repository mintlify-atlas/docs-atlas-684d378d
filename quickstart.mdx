Welcome to Pacto Lúdico! This guide will help you get up and running with the platform quickly.

## Access the Platform

Pacto Lúdico is a web-based application that you can access through any modern browser. The platform provides a comprehensive interface for managing and exploring board game collections.

<Steps>
  <Step title="Open the Application">
    Navigate to the Pacto Lúdico homepage in your web browser. You'll be greeted with the login page.
  </Step>
  
  <Step title="Login to Your Account">
    Enter your credentials to access the platform. The application supports both regular users and administrators.
  </Step>
  
  <Step title="Explore the Game Catalog">
    Once logged in, you can browse through the extensive collection of board games organized by category.
  </Step>
</Steps>

## Login Process

The login page provides a simple interface for accessing your account. Here's the HTML structure:

```html
<div id="DivIniciarSesion">
    <p>
        <input id="user" type="text" placeholder="Correo">
    </p>
    <p>
        <input id="pass" type="password" placeholder="Contraseña">
    </p>
    <p>
        <label id="RadioUserL" for="RadioUser" class="LabelUsado">Usuario</label>
        <input type="radio" name="Acceso" id="RadioUser">
        <label id="RadioAdminL" for="RadioAdmin">Administrador</label>
        <input type="radio" name="Acceso" id="RadioAdmin">
    </p>
    <p>
        <input type="button" class="btn btn-dark" onclick="inicio()" value="Ingresar">
    </p>
    <a href="app/Registrar.html">¿No se ha registrado? Registrese.</a>
</div>
```

### Authentication Logic

The login functionality is handled by JavaScript that validates credentials and redirects users based on their role:

```javascript
function inicio(){
    if(user.value=="Ivan"&&pass.value=="war"&&RadioUser.checked){
       window.location='app/MenuUser.html';
    }else if(user.value=="Ivan"&&pass.value=="war"&&RadioAdmin.checked){
        window.location='app/MenuAdmin.html';
    }
    // Error handling for invalid credentials
    if(user.value!="Ivan"){
        user.className="Error";
    }
    if(pass.value!="war"){
        pass.className="Error";
    }
}
```

<Note>
  The application distinguishes between regular users and administrators. Select the appropriate role when logging in.
</Note>

### Filtering Games by Category

The filtering functionality uses jQuery to show/hide games dynamically:

```javascript
$('.category_item').click(function(){
    var catProduct = $(this).attr('category');
    
    // Add active class to selected category
    $('.category_item').removeClass('ct_item-active');
    $(this).addClass('ct_item-active');
    
    // Hide all products with animation
    $('.product-item').css('transform', 'scale(0)');
    function hideProduct(){
        $('.product-item').hide();
    } setTimeout(hideProduct, 400);
    
    // Show filtered products
    function showProduct(){
        $('.product-item[category="'+catProduct+'"]').show();
        $('.product-item[category="'+catProduct+'"]').css('transform', 'scale(1)');
    } setTimeout(showProduct, 400);
});
```

## Switching Between Views

The user menu allows switching between Games and Events views:

```javascript
function RevelarDiv1(){
    Juegos.className="Superpuesto";
    Eventos.className=Eventos.className+" Oculto";
}

function RevelarDiv2(){
    Juegos.className=Juegos.className+" Oculto";
    Eventos.className="Superpuesto";
}
```

Buttons in the interface trigger these functions:

```html
<div class="row">
    <button onclick="RevelarDiv1()" class="boton">Juegos</button>
    <button onclick="RevelarDiv2()" class="boton">Eventos</button>
</div>
```
